2. 질문하기

  • 교육청에서 온 긴급 공문의 핵심만 빠르게 파악할 때
  • 작년 사업 결과 보고서와 올해 계획서를 비교 분석할 때
  • 여러 규정집에서 특정 내용을 한 번에 찾아내고 싶을 때

스프레드시트 자동화 마법, Apps Script

수백 명의 명단을 보며 반복적으로 체크하는 일, 이제 그만! Apps Script는 "이런 조건일 땐, 이렇게 행동해!" 라는 '마법 주문'을 한번 알려주면 평생 반복 업무를 대신 처리해줍니다.

"마법을 부려볼까요?" - 직접 체험해보세요

1. 자동화할 작업 (스프레드시트)

2. AI에게 마법 주문(코드) 요청하기

"성적이 'A'이고 봉사 시간이 20 이상인 행의 배경색을 초록색으로 바꿔줘."

마법 실행 버튼을 누르면 AI가 만든 주문(코드)이 여기에 나타납니다.

이럴 때 활용해 보세요!

  • 특정 조건(성적, 출석률 등)을 만족하는 학생 명단 자동으로 분류하기
  • 납부 기한이 지난 항목에 자동으로 색깔이나 표시 남기기
  • 버튼 하나로 명단에 있는 사람들에게 이메일 대량 발송하기 (심화)

"글을 써볼까요?" - 직접 체험해보세요

1. 작성할 글 종류 선택하기

2. AI 작가에게 보낼 요청서 (프롬프트)

3. AI 작가가 작성한 초안

글쓰기 요청 버튼을 누르면 AI가 작성한 글이 여기에 표시됩니다.

이럴 때 활용해 보세요!

  • 모든 종류의 공문, 안내문, 행사 기획서 초안 작성하기
